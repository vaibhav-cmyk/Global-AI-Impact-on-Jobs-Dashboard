📊 Global AI Impact on Jobs – Power BI Dashboard
📌 Project Overview

This project analyzes the impact of Artificial Intelligence on global jobs using an interactive Power BI dashboard.

The dashboard highlights how AI is influencing:

Job automation risk

Salary changes across industries

Skill demand growth

AI adoption worldwide

Workforce reskilling needs

The goal is to help understand which industries and job roles are most affected by AI transformation.
